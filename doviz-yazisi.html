Web Siteme Nasıl Canlı Döviz Bandı Ekledim?

📅 14 Mart 2026 ⏱️ 4 Dakika Okuma

Bir web sitesini canlı kılan en önemli detaylardan biri, dış dünyadan gelen dinamik verilerdir. Portfolyo sitemi geliştirirken, hem finansal verileri takip etmeyi sevdiğim hem de teknik bir beceri sergilemek istediğim için navigasyon kısmına bir döviz bandı eklemeye karar verdim.

1. Veri Kaynağı: API Seçimi

Canlı verileri çekmek için ExchangeRate-API servisini kullandım. Ücretsiz planı sayesinde USD, EUR ve TRY kurlarına kolayca erişebiliyoruz. İlk adım olarak Fetch API ile bu verilere ulaştım.

async function dovizGetir() {
    const response = await fetch('https://api.exchangerate-api.com/v4/latest/USD');
    const data = await response.json();
    console.log(data.rates.TRY); // Dolar kurunu verir
}

2. Kayan Yazı Efekti (CSS Animation)

Verilerin ekranın sağından soluna doğru akması için @keyframes animasyonunu kullandım. Yazıların birbirini takip etmesi için içerik genişliğini white-space: nowrap ile sabitleyip sürekli dönen bir döngü oluşturdum.

@keyframes kaydir {
    0% { transform: translateX(100%); }
    100% { transform: translateX(-150%); }
}

3. Sonuç

Artık sitemin üst kısmında hem dolar ve euro kurlarını hem de popüler kripto para verilerini canlı olarak görebiliyorum. Bu sistem her dakika kendini yenileyerek güncel kalıyor.
